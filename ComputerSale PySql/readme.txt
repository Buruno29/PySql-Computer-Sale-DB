0- Linking SQL and Python | this video worked as installation help, but it may be necessary to use a different version of the video: https://www.youtube.com/watch?v=3vsC05rxZ8c . 
1- Go to the "0create db" program and run it. 
2- Create the tables in the codes ending in "tablecreate" (order: customers, product, orders, orderdetails, Payments). 
3- Inserting data: To make things easier, we left another text file "registros.txt" with customer, product and similar data, in the codes ending in "insert". 
4- In "0 main", it is possible to query purchases and see the tables created, but other queries are available in codes ending in "show".
!- The payments table does not have an insertion yet, but with the right data, manual work would be possible.
